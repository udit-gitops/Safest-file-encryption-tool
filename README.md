This is an offline desktop app to encrypt & decrypt files securely using modern AES encryption.

Built with:
- Python 
- Tkinter + ttkbootstrap (modern responsive UI)
- PyCryptodome (AES encryption)

Features:

- Works completely offline
- Simple 2-screen UI:
    - Choose Encrypt / Decrypt
    - Enter file + password
- Generates .encrypted files
- Decrypts back to original
- Safe and fast
